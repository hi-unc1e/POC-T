特点
---
* 支持多线程/Gevent两种并发模式  
* 极简式脚本编写，无需参考文档  
* 内置脚本扩展及常用PoC函数  
* 支持第三方搜索引擎API(已完成ZoomEye/Shodan/Google/Fofa免费版)  


依赖
---
* Python 2.7 / 3.9
* pip
```markdown
python2 ./shodan-python-1.27.0/setup.py install

sudo yum -y install python-devel
```
